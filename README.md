# OpenCart Testing Project

## 📌 Overview
This project demonstrates manual and automation testing on OpenCart demo website.

## 🧪 Manual Testing
- Designed and executed 80+ test cases
- Covered Login, Registration, Cart, Checkout
- Documented results in Excel

## 🐞 Bug Reporting
- Created detailed bug reports with severity and steps
- Linked failed test cases with bugs

## 🤖 Automation Testing
- Automated test scenarios using Selenium with Java
- Used TestNG and Maven

## 🔧 Tools
- Selenium WebDriver
- Java
- TestNG
- Maven
- Postman
- Jira (basic usage)
